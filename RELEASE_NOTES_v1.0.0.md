## English

Beginner-friendly first release for opening Twitch streams in PotPlayer at source/original quality.

Included:
- Tampermonkey right-click menu for Twitch stream pages and cards
- twitchpotplayer:// Windows protocol installer
- Direct CMD launcher: open-twitch-source.cmd
- PowerShell source-quality launcher using Streamlink
- QUICK_START.md in English, Korean, and Russian

Recommended download:
- twitch-potplayer-720p-fix-beginner.zip

Requirements:
- Windows
- PotPlayer 64-bit
- Streamlink
- Chrome or Chromium browser
- Tampermonkey, only for browser right-click menu usage

Notes:
- If a stream is only broadcast at 720p, source quality is also 720p.
- Public proxy endpoints can temporarily fail.
- This project does not use Twitch cookies or auth-token.

## Русский

Первый релиз для начинающих: открывает Twitch-стримы в PotPlayer в исходном качестве (source/original quality). Подходит для случаев, когда Twitch в PotPlayer открывается только в 720p.

Включено:
- Меню правой кнопки мыши Tampermonkey для страниц и карточек Twitch
- Установщик Windows-протокола twitchpotplayer://
- Прямой CMD launcher: open-twitch-source.cmd
- PowerShell launcher для исходного качества через Streamlink
- QUICK_START.md на английском, корейском и русском

Рекомендуемая загрузка:
- twitch-potplayer-720p-fix-beginner.zip

Требования:
- Windows
- PotPlayer 64-bit
- Streamlink
- Chrome или Chromium-браузер
- Tampermonkey, нужен только для меню правой кнопки мыши в браузере

Важно:
- Если стример вещает только в 720p, source тоже будет 720p.
- Публичные proxy-серверы могут временно не работать.
- Проект не использует Twitch cookies или auth-token.
- Не скачивайте один .cmd файл отдельно; используйте весь ZIP целиком.
